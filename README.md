# Taemin's dotfiles

Collection of configuration files for bash, git, fzf, ssh and so on.

# Installation

Just clone this repository and load bootstrap.sh like below:

`git clone https://github.com/taeminhong/dotfiles.git && cd dotfiles && . bootstrap.sh`
